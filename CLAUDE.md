# LOCAL NOMAD — Local AI Agent

Проект управляется фреймворком **KAIF** (см. `KAIF_FRAMEWORK.md`).

**Перед каждой задачей читай канон проекта — [AGENT_GUIDE.md](AGENT_GUIDE.md)** (правила, карта,
команды, конвенции, чек-лист перед задачей), затем [STATUS.md](STATUS.md) (текущее состояние и «где
продолжить»).

Главный мыслительный принцип — `PHILOSOPHY.md` (ПРОСТОТА: KISS + Оккам). Починка дефектов — по
`BUG_FIXING_FRAMEWORK.md`. Рабочий язык проекта — русский.
