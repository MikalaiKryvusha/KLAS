## Контекст (30 секунд)

Ревизия OSS-поля (июль 2026) показала: **все крупные блоки ассистента собираются из готового** —
русский STT (sherpa-onnx + GigaAM-v3, MIT), русский TTS с клонированием (CosyVoice 3 / Chatterbox V3 /
F5-TTS_RUSSIAN), каркас голосового каскада (Pipecat), управление Windows (Windows-MCP). Главная
новость: **OpenClaw** (экс-Clawdbot) — 383k ⭐, Node/TS, MIT — готовое «ядро персонального ассистента»
(навыки-CRUD в формате SKILL.md, память, каналы, cron, Android-нода) с поддержкой локальных LLM.
Технические обратимые выборы (конкретный STT/TTS/каркас) я решу сам по дефолтам исследования — здесь
только необратимое/владельческое.

### Q1. Ядро ассистента — OpenClaw или свой тонкий Gateway?

Это выбор формы всего эпика (plans/07 Фаза 2): взять гигантский готовый модуль или собрать своё ядро
из модулей поменьше.

- **A) (рекомендую)** **Пилот OpenClaw → решение на фактах.** Разворачиваю OpenClaw в tailnet на
  llama-server, гоняю наши agent-bench-задачи в его цикле, меряю «вывозит ли» локальная модель его
  агентность; параллельно Фаза 1 плана 07 («скелет разговора») из модулей — она полезна при любом
  исходе. Выбор ядра — отдельным решением после пилота. *Почему: обратимо, дёшево, решение на замерах,
  ничего не блокирует.*
- **B)** Сразу **свой тонкий Gateway** (текущий plans/07), OpenClaw не трогаем. *Максимум контроля и
  приватности, минимум чужого кода; писать больше.*

  локальных моделей 9–35B в его цикле не подтверждён.*

### Q4. Полномочия агента над Windows (когда дойдём до PC-контроля)

Windows-MCP даёт агенту мышь/клавиатуру/запуск программ/shell. Это мощно и опасно; объём полномочий —
решение владельца (безопасность).

- **A) (рекомендую)** **Ступенчато с allow-list:** старт = только чтение экрана (UIA) + запуск программ
  из белого списка; мышь/клавиатура/произвольный PowerShell — отдельными явными включениями позже,
  после обкатки. *Почему: стабильность и безопасность выше скорости — твой же приоритет.*
- **B)** Сразу полный доступ (мышь/клавиатура/shell) — «как у Тони Старка», риски принимаю.
