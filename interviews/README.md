# `interviews/` — решения уровня владельца

Интервью, которые агент проводит с человеком, чтобы закрыть решения, которые он **не должен** принимать
один — UI/UX, серьёзные технические развилки, бренд/видение/приоритеты. Закрытые вопросы A/B/C/D с
рекомендацией первым пунктом; человек отвечает **прямо в документе**. Один
`interview_NNN_<topic>.md` на интервью.

**Человеку (владельцу):** когда агент завёл интервью — он ждёт **тебя**. Заполни поля «**Ответ:**»
прямо в документе (выбери A/B/C или напиши своё в D). Здесь фиксируются и сохраняются твои судьбоносные
решения.

**ИИ-агенту:** заводи интервью только для по-настоящему владельческих развилок (скилл: `/interview`).
Варианты — **A/B/C/D**: **A** — всегда выбор, дистиллированный через `PHILOSOPHY.md`
(простейший/эффективнейший), с пометкой **(рекомендую)**; **D** — всегда «свой ответ» владельца.
Сначала сделай подготовительную работу, держи 1–5 вопросов, затем поставь паузу и дай владельцу
ответить. Всё, что дёшево откатить, — решай сам.
