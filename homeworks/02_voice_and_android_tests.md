## Часть 1 — Выбор голоса KLAS (~2 мин, Г1 · plans/11)

Прослушай кандидатов (динамики включи; фразу можно любую свою):

```powershell
node tools/voice-say.mjs "Привет, Криник! Я — КЛАС." --play --voice aidar
node tools/voice-say.mjs "Привет, Криник! Я — КЛАС." --play --voice eugene
node tools/voice-say.mjs "Привет, Криник! Я — КЛАС." --play --voice baya
node tools/voice-say.mjs "Привет, Криник! Я — КЛАС." --play --voice kseniya
node tools/voice-say.mjs "Привет, Криник! Я — КЛАС." --play --voice xenia
```

(aidar/eugene — мужские, baya/kseniya/xenia — женские; это ВРЕМЕННЫЙ голос до клонов Вихрова/Joi из Г5.)

## Часть 2 — Живой голосовой диалог с ядром (~3 мин, Г3 · plans/13)

```powershell
node tools/voice-talk.mjs
```

- Enter — начать говорить, Enter — закончить; ответ придёт голосом (ход ~20–25 с, ядро думает дольше всех).
- Скажи 2–3 связные реплики (например: «запомни: я люблю кофе» → «что я люблю?») — проверка памяти сессии.
- Микрофон по умолчанию — «Микрофон (NVIDIA Broadcast)»; если у тебя другой:
  `node tools/voice-talk.mjs --device "имя из ffmpeg -list_devices true -f dshow -i dummy"`.
- Выход — пустая реплика (два Enter подряд) или Ctrl+C.

## Часть 3 — Android-нода OpenClaw (Г4 · plans/14, делаем ВМЕСТЕ с агентом)

Твоя часть — только телефон; агент ведёт по шагам в чате:

1. **Tailscale на телефон** (если ещё нет): установить из Play Store, войти своим аккаунтом —
   телефон появится в tailnet.
2. **Нода OpenClaw**: скажи агенту «ставим ноду» — он подготовит актуальную инструкцию установки
   (приложение-нода из экосистемы OpenClaw) и данные подключения к гейтвею
   (ws://100.80.125.66:18789 + токен из `caddy/PASSWORD.local.txt`).
3. Проверка: в Voice-вкладке ноды сказать фразу — ответ должен прийти голосом Silero С ХОСТА
   (наш TTS уже подключён к ядру). Wake word на Android апстрим пока отключил — старт кнопкой.
