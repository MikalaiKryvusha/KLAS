# `homeworks/` — задачи от агента человеку

Задачи, которые агент просит сделать **человека**, — то, что сам не может из-за своей цифровой,
бестелесной природы: проверить на реальном железе, действовать в физическом мире, воспользоваться
аккаунтом/учёткой, доступной только человеку, купить, понаблюдать офлайн. Каждый документ описывает
задачу конкретными шагами для человека и собирает обратно его наблюдения и результаты. Один
`NN_<name>.md` на задачу.

**Человеку (владельцу):** когда агент завёл домашку — ему нужна рука в физическом/офлайн-мире. Пройди
шаги и запиши, что наблюдал, обратно в документ — агент прочитает твои заметки и продолжит.

**ИИ-агенту:** когда заблокирован тем, что может сделать только человек-с-телом, — не стой: напиши
сюда домашку с ясными, минимальными, пронумерованными шагами и местом для результатов человека, затем
продолжай другую работу. Когда человек отчитался — учти результаты и поставь файлу тег `DONE`
(`git mv NN_x.md NN_DONE_x.md`).
