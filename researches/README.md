# `researches/` — база знаний по большим сложным вопросам

Заметки и дистиллированные находки по крупным трудным вопросам — write-up'ы исследований, накопленное
ноу-хау, важная справочная информация, которая потребовала реальных усилий и не должна испариться.
Один `NN_<name>.md` на тему исследования.

**Человеку (владельцу):** место, где найти глубокий фон за решениями — почему выбран подход, что
узнали о трудной проблеме. Можешь посеять тему, которую хочешь исследовать.

**ИИ-агенту:** когда вопрос достаточно крупный, чтобы его находки пережили текущую задачу, — пиши
исследовательскую заметку сюда (сырые источники → анализ → выводы/гипотезы). Ссылайся на неё из
бага/плана/идеи, которые её мотивировали (DRY — не переисследуй). Исследовательская заметка — **живой
справочник**, не закрываемая задача: тег `DONE` не ставится; обновляй по мере роста понимания.
