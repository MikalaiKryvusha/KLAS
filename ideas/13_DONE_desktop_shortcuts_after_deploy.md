# Idea 13 — ярлыки на Рабочем столе после деплоя (Run / Stop / Control Panel)

> Источник: **владелец** (в чате, 2026-07-04).
> Статус: ✅ ГОТОВО (2026-07-04). `tools/install-desktop-shortcuts.ps1` создаёт на Рабочем столе
> Run KLAS / Stop KLAS / KLAS Control Panel (иконка Кот Криник); тихий раннер `tools/klas-run.vbs`.
> Проверено — 3 ярлыка создаются. Вызов из `deploy.mjs` — в рамках идеи 12 (умный установщик).

## Суть

После разворачивания KLAS на Рабочем столе пользователя должны автоматически появиться **три ярлыка**
с иконкой «Кот Криник»:

| Ярлык | Действие | Скрипт |
|-------|----------|--------|
| **Run KLAS** | Поднять весь стек (docker + llama-swap + funnel + трей) | `tools\klas.ps1 -Action up` |
| **Stop KLAS** | Погасить весь стек | `tools\klas.ps1 -Action down` |
| **KLAS Control Panel** | Открыть веб-пульт локально | открыть `http://localhost/` |

- «KLAS Control Panel» просто открывает локалхост-вебку пульта в браузере по умолчанию.
- Все три — с иконкой Кот Криник (`logo/*.ico`), тихий запуск без окна консоли (как уже сделано для
  трея через `.vbs`-обёртку в идее 10).

## Что уже есть (переиспользовать)

- `tools/klas.ps1 -Action up|down` — подъём/гашение стека (идея 10).
- `tools/install-klas-shortcut.ps1` — уже создаёт ОДИН ярлык `KLAS.lnk` (трей). Расширить до трёх
  отдельных ярлыков (Run / Stop / Control Panel), либо добавить рядом.
- `tools/klas-control.vbs` — образец тихого запуска PowerShell без окна.

## Реализация (набросок)

1. Скрипт `tools/install-desktop-shortcuts.ps1` создаёт 3 `.lnk` на Рабочем столе (и, опционально, в
   меню Пуск): Run KLAS → `klas.ps1 -Action up` (через vbs, скрыто), Stop KLAS → `-Action down`,
   KLAS Control Panel → `http://localhost/` (ShellExecute URL, откроется в браузере). Иконка — `logo/*.ico`.
2. **Вызывать этот скрипт в конце `deploy.mjs`** (идея 03/12) — чтобы ярлыки появлялись автоматически
   «под ключ» после установки. Идемпотентно: повторный деплой не плодит дубли.
3. Учесть classifier-ограничение: автозапуск при входе (persistence) ставит ВЛАДЕЛЕЦ отдельной командой
   (идея 10). Создание ярлыков на Рабочем столе — не persistence, его деплой делать может.
