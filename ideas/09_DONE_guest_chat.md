# Idea 09 — Гостевой чат для родных (ChatGPT-подобный, с аккаунтами)

> Источник: **владелец** (2026-07-03). Оформлено агентом; смыслы сохранены.
> Статус: ✅ ГОТОВО (2026-07-04) — подтверждено владельцем. Open WebUI поднят (личные аккаунты,
> авторизация, приватные чаты), доступен локально `http://localhost:3080/` и через funnel `:8443`.

## Суть

На гостевом пульте есть llama-swap, но для **неподготовленных родных**, которым нужен опыт ChatGPT,
это непонятно и сложно. Нужен удобный **гостевой чатик** — простой, привычный, желательно с
**личными аккаунтами и авторизацией**, чтобы родные не читали чаты друг друга. Прикрутить к KLAS и
пробросить на homepage.

## Решение (выбор владельца)

**Open WebUI** (`open-webui/open-webui`) — ChatGPT-подобный self-hosted чат:
- Личные аккаунты + авторизация, изоляция диалогов на уровне пользователя (приватные чаты), роли
  admin/user. Самый зрелый мультиюзер среди аналогов (2026).
- Подключается к нашему **llama-swap** по OpenAI-совместимому API — видит все модели KLAS.
- Один docker-контейнер; порт 3080 → проброс на homepage и через Caddy.

Сравнение с AnythingLLM (владелец спрашивал): AnythingLLM — «RAG-и-агенты с чатом сбоку»
(workspace-центричный, мультиюзер менее зрелый); Open WebUI — «чат с RAG сбоку», зрелый мультиюзер и
приватные чаты. Под «гостевой ChatGPT для родных» побеждает **Open WebUI** (знания у нас уже
закрыты kiwix + openzim-mcp, RAG-преимущество AnythingLLM не нужно).

## Доступ (выбор владельца)

Удалённо через **Tailscale Funnel** (публичный URL, родным без установки Tailscale; защита — логин
Open WebUI). ⚠️ Обнаружено: Funnel не открывается извне, пока на ПК активен **NordVPN** (NordLynx
перехватывает трафик) — см. заметку в реализации; для работы Funnel NordVPN нужно полностью выйти.

## Реализация

- Docker-сервис `open-webui` в `docker-compose.yml` (порт 3080, том `open-webui/data`,
  `OPENAI_API_BASE_URL=http://host.docker.internal:8080/v1`, `WEBUI_AUTH=true`).
- Проброс на пульт homepage (карточка «Чат для родных») + маршрут в Caddy (`/chat`) под Funnel.
- Первый зарегистрированный пользователь = админ (владелец); остальным — обычные аккаунты.
