# Idea 04 — Забыть LOCAL_NOMAD: вычистить из проекта следы миграции

> смыслы и посылы сохранены дословно.
> Статус: ✅ ОДОБРЕНА по факту авторства владельца.

## Суть

Для появления этого проекта выполнялась миграция из директории пользователя `LOCAL_NOMAD` в `F:\KLAS`.
Это была **разовая история** — дальше проект развивается как сформированная идея:

> Krinik Local Agent System (KLAS) — a self-hosted AI ecosystem powered by a local LLM on gaming GPU.
> It features autonomous AI agents, a web dashboard for control, and a localized Wikipedia-style
> knowledge base for total data privacy and offline efficiency. KAIF-powered framework for homelab
> enthusiasts.

Соответственно, всё, что касалось миграции «LOCAL_NOMAD → KLAS», больше неважно и не нужно — это
нужно забыть и стереть, а сосредоточиться только на реализации видения KLAS: **легко
распаковывающийся фрейм, настраивающий локальную умную и быструю агентскую ИИ-систему на машине
пользователя**.

## Задача

Вычистить из проекта **ВСЁ**, что касалось миграции «LOCAL_NOMAD → KLAS»:

- из всех документов, планов, багов;
- скрипты миграции — удалить;
- планы миграции — удалить;
- ключевые md-документы KAIF-фреймворка — очистить от любых знаний, инструкций и памяти о процессе
  миграции.

Это больше неважно и только засоряет проект и репозиторий KLAS. Во всех документах фокусируемся на
глобальной идее и цели проекта KLAS.
