# Идея 10 — Управление KLAS из трея Windows (старт/стоп) + пуш-уведомление

**Статус:** ✅ реализовано 2026-07-04 (кроме автозапуска — его ставит владелец одной командой, см. ниже).
**Автор идеи:** владелец. **Фаза:** 5 (пульт/удобство).

---

## Что просил владелец

- В области уведомлений Windows 11 — своя иконка KLAS (Кот Криник).
- По правой кнопке — меню с кнопками **[Open KLAS control panel]** и **[Stop KLAS and exit]**:
  - первая открывает в браузере локальный пульт KLAS;
  - вторая останавливает все контейнеры и все службы KLAS.
- Внутри проекта — exe/ps1/bat, который одним запуском поднимает весь KLAS, + ярлык Windows с
  иконкой котика KLAS.
- При запуске KLAS — пуш-уведомление Windows «KLAS запущен и работает».
- В homepage — ссылка на репозиторий KLAS в GitHub; при анонимной установке этой ссылки нет.
- Иконка внутри уведомления — котик (не стандартный синий info-круг).

---

## Как сделано

**Контроллер `tools/klas.ps1`** (один скрипт, три действия):
- `-Action tray` (по умолчанию) — иконка «Кот Криник» (`logo/homepage.ico` — favicon, не баннер) в
  области уведомлений; правый клик → меню **Open KLAS control panel** (открывает `http://localhost:3005/`) /
  **Stop KLAS and exit**; двойной клик по иконке — открыть пульт. При старте поднимает весь стек и
  показывает пуш «KLAS запущен и работает» (`ToolTipIcon.None` — без синего круга, с иконкой-котом).
  Защита от повторного запуска (mutex) — одна иконка даже при двойном клике по ярлыку.
- `-Action up` — поднять весь KLAS: docker-сервисы (kiwix/open-webui/homepage/caddy) + llama-swap
  (тихо, без окна) + Tailscale Funnel. Идемпотентно. Это и есть «один запуск поднимает весь KLAS».
- `-Action down` — остановить всё: funnel off → `docker compose stop` → убить llama-swap/llama-server.

**Тихий запуск без окна:** `tools/klas-control.vbs` (`wscript` → `powershell -Sta ... klas.ps1 -Action tray`).

**Ярлык с котиком:** `tools/install-klas-shortcut.ps1` создаёт `KLAS.lnk` (иконка `logo/homepage.ico`)
в корне проекта и на Рабочем столе; цель — скрытый запуск трея. `KLAS.lnk` — вне git (машинно-зависимый).

**Ссылка на репозиторий в homepage:** закладка «Исходный код KLAS (GitHub)» в
`homepage/config/bookmarks.yaml` (группа «Проект»). При анонимной установке `tools/anonymize.mjs`
удаляет этот блок целиком (homepage в SKIP_DIRS, поэтому вырезается отдельным шагом).

## Автозапуск при входе в систему — ставит ВЛАДЕЛЕЦ (одна команда)

Агенту classifier запрещает создавать persistence-механизмы. Поэтому автозапуск ставит владелец:

```
powershell -File F:\KLAS\tools\install-autostart.ps1
```

Скрипт создаёт задачу Планировщика **«KLAS»** (триггер: вход в систему) → тихо поднимает весь стек +
трей «Кот Криник» + пуш. Он же снимает прежнюю задачу-предшественника «KLAS llama-swap» (её роль
поглотил трей). Удалить: `Unregister-ScheduledTask -TaskName "KLAS" -Confirm:$false`.

## Проверено (2026-07-04)

- `-Action up` идемпотентно поднимает стек; `-Action down` гасит всё; цикл down→up восстанавливает
  контейнеры, llama-swap (/health 200), funnel и пульт через funnel (200).
- Трей запускается без окна консоли, single-instance mutex не плодит копии.
- Ярлыки созданы (корень + Рабочий стол), иконка — favicon-кот.
- homepage отдаёт закладку на репозиторий (`/api/bookmarks` 200); anonymize чисто вырезает блок.
