# План 15 — Голосовой тракт Г5: «Свой голос» (клоны + персоны Jarvis/Joi)

> Фаза Г5 голосового блока — финальная шлифовка «как с живым». Зонтик —
> [plans/07](07_jarvis_gateway_and_client.md). Решения: интервью 004 Q3 (слепой A/B клонов:
> CosyVoice 3 / Chatterbox V3 / F5-TTS_RUSSIAN; до них — Silero), интервью 003 Q2 (персоны:
> дефолт Jarvis, обе отзываются на оба имени; голос Joi — девичий, бархатный).
> Статус: 🔲 не начат (после Г3; требует 🏠 домашку владельца — референс-сэмплы голосов).

## Цель

Ассистент говорит **выбранным владельцем голосом** (клон Вихрова для Jarvis / бархатный женский для
Joi), переключение персон работает; поверх скелета — естественность (стриминг TTS по предложениям,
первый звук быстрее).

## Шаги

- [ ] 🏠 **Домашка владельца — референс-сэмплы:** чистые записи голоса Вихрова (Jarvis) и кандидаток
      Joi (10–30 с каждая, без музыки/шума). Без сэмплов фаза стоит — напомнить владельцу.
- [ ] **Стенд A/B** `tools/voice-ab.mjs`: одинаковый набор фраз (приветствие, число/дата, техфраза,
      эмоциональная) синтезируется тремя движками zero-shot клоном по сэмплу; wav-файлы в слепом
      порядке (манифест соответствий отдельно) — владелец слушает и выбирает (🏠).
- [ ] **Три кандидата** (веса локально, каждый в своём venv/подпапке `F:\KLAS\voice\`):
      CosyVoice 3 (Apache-2.0) · Chatterbox V3 (MIT) · F5-TTS_RUSSIAN (NC-веса — ок для личного
      использования, в продукт не тащить). Замер: качество просодии (уши владельца), RTF на нашей
      GPU/CPU, стабильность ударений.
- [ ] **Интеграция победителя** в тракт (Г1-интерфейс `voice-say` получает `--voice jarvis|joi`),
      Silero остаётся fallback-голосом.
- [ ] **Персоны** (инт. 003 Q2, идея 05): дефолт Jarvis; обе на оба имени («коллега отдыхает…»);
      переключение голосом; голос меняется вместе с персоной.
- [ ] **Естественность:** стриминг TTS по предложениям (первый звук раньше), интонационные пресеты,
      при выходе на телефон — Pipecat/WebRTC-сессия (решение по метрикам Г4).

## Проверка (критерий готовности)

Слепой A/B пройден владельцем, выбор зафиксирован здесь; «Jarvis, …» отвечает голосом Вихрова,
«Joi, …» — женским бархатным; fallback Silero работает при недоступности клона.

## Риски

- Лицензии: F5-TTS_RUSSIAN — NC (только личное); следить, чтобы NC-веса не попали в
  публикуемые артефакты KLAS.
- Клоны тяжелее Silero (GPU/латентность) — держать Silero дежурным, клон — по требованию
  (llama-swap-принцип «спит, пока не позовут» применим и тут).
